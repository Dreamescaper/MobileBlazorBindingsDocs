Architecture of Maui Blazor Bindings

Important

This project is an experimental release. We hope you try out Experimental Maui Blazor Bindings and provide feedback at https://github.com/Dreamescaper/BlazorBindings.Maui.

Experimental Maui Blazor Bindings consists of two main libraries:

  1. BlazorBindings.Core: A library that is used to abstract arbitrary native UI libraries to work with the Blazor UI model. This library contains implementations of an adapter system that maps between native UI elements and Blazor components.
  2. BlazorBindings.Maui: A library that references the Core library and adds specific bindings for Xamarin.Forms elements to enable building native apps for the Android and iOS mobile platforms.
